Aggregate pass criterion (D-14):
- `rerank_top3_hits = sum(1 for case in cases if any(gold_id in top3(rerank_retriever, case.question) for gold_id in case.gold_chunk_ids))`
- `dense_only_top3_hits = sum(1 for case in cases if any(gold_id in top3(dense_only_retriever, case.question) for gold_id in case.gold_chunk_ids))`
- `assert rerank_top3_hits > dense_only_top3_hits, "Reranker did not measurably improve top-3 hit rate"`
- `assert rerank_top3_hits >= 5, "Rerank pipeline did not hit gold in top-3 on at least 5 cases"`
